# FUTURE_ML_03 - Resume Screening System

## Objective
Build an ML system to automatically screen and rank 
resumes based on a given job role.

## Dataset
Resume Dataset - 2484 resumes, 25 categories

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Kaggle Notebook
[View Notebook](https://www.kaggle.com/code/moni35/notebook5788777f)

## Results
- Screened 2484 resumes for ML Engineer role
- Ranked top 10 candidates by skill match
- Identified skill gaps for top candidate
- Match Score: 46.15%

## Skills Matched
python, data science, pandas, numpy, sql, statistics

## Missing Skills (Skill Gap)
machine learning, deep learning, tensorflow, 
pytorch, sklearn, neural network, nlp
